# **App Name**: AnishDev Folio

## Core Features:

- Dynamic Content Display: Showcase Anish Kushwaha's professional details by extracting personal info (name, title, DOB, nationality, gender, phone, email, address, social media links) from the resume. This includes a full-screen hero section, an 'About Me' bio highlighting 'best learning skills in Technology and Developments,' 'Quick learner,' and 'AI enthusiast,' categorized skills (Mobile Dev, Frontend, Backend, Programming), an educational timeline (SSC, HSC, BCA), language proficiency (Hindi, English, Gujarati), and extra skills (Video editing, Logo design, AI tools).
- Interactive Project Showcase: Display projects fetched from Anish's resume links as interactive cards with live preview buttons, tech stack tags, and engaging hover animations (scale + glow) to highlight featured work and provide an attractive, well-graphic UI.
- Firebase-powered Contact Form: Enable visitors to directly contact Anish Kushwaha via a working form, with submissions securely managed and stored using Firebase's Firestore for data persistence.
- Generative AI Introduction Tool: Implement a subtle AI assistant floating icon that, when activated, introduces Anish Kushwaha and his skills to visitors using a personalized, generated script, demonstrating the developer's engagement with AI tools.
- Elegant Animations & Micro-interactions: Incorporate smooth transitions, an animated typing effect in the hero section, animated background elements, interactive counters for stats, glowing hover effects for cards and interactive elements, scroll-triggered animations for content reveals (e.g., education timeline), and seamless page transitions for a fluid user experience. An animated cursor/hover glow will enhance micro-interactions, all powered by Framer Motion.
- Responsive Light Glassmorphism UI: Ensure a mobile-first, fully responsive design featuring a vibrant light UI with glassmorphism elements, dynamic gradient accents, and soft shadows for a clean, minimal yet premium feel (inspired by Apple + modern SaaS sites). Includes an optional dark/light mode toggle for optimal viewing on any device.
- Basic SEO Configuration: Implement fundamental SEO best practices within the Next.js application to improve discoverability by search engines for recruiters and potential clients, ensuring the website is high-converting.

## Style Guidelines:

- A vibrant, multi-color light-themed palette designed for maximum visual appeal, attracting attention without being harsh on the eyes. The primary background will be a crisp, clean off-white (#F8F9FA) or a very subtle light grey. This is complemented by a dynamic range of gradient accents combining electric blues (#007bff), energetic magentas (#E83E8C), fresh teals (#20C997), and soft oranges (#FFC107). These vibrant yet harmonious combinations will be used strategically for interactive elements, glowing hover effects, background details, and glassmorphism overlays to ensure a highly attractive and engaging user experience. Dark, high-contrast text (#212529) will ensure excellent readability.
- 'Inter' (sans-serif) for its modern, clean, and objective aesthetic, providing excellent readability across all text sizes and maintaining a premium, minimal feel. This single font handles all textual content consistently.
- Minimalist, line-art style icons with subtle glowing effects on hover. Animated icons will be used for language proficiencies and extra skills to add visual interest, complementing the light glassmorphism theme. A custom floating icon will represent the AI assistant.
- Mobile-first, fluid layout using CSS Grid and Flexbox for responsiveness, with generous whitespace for a clean, premium presentation. Key sections, such as projects and skills, will be presented as 'glassmorphism' styled interactive cards. Features a full-screen hero section, a sticky navbar with a blur effect, and a timeline layout for education.
- Smooth, purposeful animations primarily driven by Framer Motion. This includes an animated typing effect in the hero section, animated background elements (particles/gradient/blobs), interactive counters for statistics (skills/projects/experience), glowing hover effects for cards and interactive elements, scroll-triggered animations for content reveals (e.g., education timeline), and seamless page transitions for a fluid user experience. An animated cursor/hover glow will enhance micro-interactions.
